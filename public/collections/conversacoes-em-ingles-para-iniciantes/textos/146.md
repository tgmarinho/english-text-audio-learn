## English

A: Veronica, please, send this envelope to this address.

B: Sorry, what did you say?

A: I said, send this envelope to this address.

B: I got it now.

A: Did you send the envelope?

B: What envelope?

A: The envelope I told you to send before.

B: Oh, I’m sorry. I forgot to send the envelope.

A: Are you kidding?

B: I’m so sorry for that, Mr. Robert.

## Português

A: Veronica, por favor, envie este envelope para este endereço.

B: Desculpe, o que você disse?

A: Eu disse, envie este envelope para este endereço.

B: Eu entendi agora.

A: Você enviou o envelope?

B: Que envelope?

A: O envelope que eu disse para você enviar antes.

B: Ah, desculpe. Esqueci-me de enviar o envelope.

A: Você está brincando?

B: Sinto muito por isso, Sr. Robert.
