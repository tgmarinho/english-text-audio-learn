## English

Work is important. Everyone should work. We work to earn money. We work to stay fit. We work to be successful. We work to achieve our goals. Hard work always pays off. Work involves mental and physical efforts. Working is good. We say, “Work is worship”. But working all the time is not good. One must always take some time out for themselves. One must relax and then start working again. We must not be stressed about our work. We should be productive. Working all the time drains all our energy. We must take out some time for our family. It is equally important for us to play. It keeps us healthy. One should be honest about his work.

## Português

O trabalho é importante. Todos deveriam trabalhar. Trabalhamos para ganhar dinheiro. Trabalhamos para ficar em forma. Trabalhamos para ter sucesso. Trabalhamos para atingir nossos objetivos. O trabalho duro sempre compensa. O trabalho envolve esforços mentais e físicos. Trabalhar é bom. Dizemos: “Trabalho é adoração”. Mas trabalhar o tempo todo não é bom. É preciso sempre reservar algum tempo para si. É preciso relaxar e começar a trabalhar novamente. Não devemos ficar estressados com o nosso trabalho. Devemos ser produtivos. Trabalhar o tempo todo esgota todas as nossas energias. Devemos reservar algum tempo para nossa família. É igualmente importante para nós se divertir. Isso nos mantém saudáveis. Deve-se ser honesto sobre seu trabalho.
